# Lumiere - Libreria Digital

Proyecto Final de Programacion Web.

## Descripcion
Sitio web de libreria digital con catalogo de libros, sistema de compras y gestion completa (CRUD).

## Tecnologias utilizadas
- HTML5
- CSS3 + Bootstrap 5
- JavaScript
- PHP
- SQLite (PDO)
- Git + GitHub

## Paginas
- index.html - Pagina de inicio
- catalogo.php - Catalogo de libros con filtros y busqueda
- agregar.php - Formulario para agregar libros
- editar.php - Editar libros existentes
- eliminar.php - Eliminar libros
- comprar.php - Sistema de compra que descuenta stock

## Funcionalidades
- Catalogo dinamico de libros por categoria y subcategoria
- CRUD completo de libros
- Busqueda por titulo y autor
- Sistema de compra con control de stock
- Validacion de formularios con JavaScript
